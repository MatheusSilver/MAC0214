# Log de Competições

Este diretório registra os resultados e participações em competições de programação durante o semestre da disciplina MAC0214.

## Estrutura

Cada competição possui seu próprio registro contendo:
- Certificados de participação
- Resultados e classificações

## Formato dos Registros

### Competições Individuais

```
competicao-nome-data/
├── certificado.pdf (Se aplicável)
└── resultado.md
```

### Competições em Equipe

```
competicao-nome-data/
├── certificado.pdf (Se aplicável)
└── resultado-equipe.md
```
