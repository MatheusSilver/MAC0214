# Soluções

Este diretório contém as soluções dos problemas resolvidos durante a disciplina MAC0214, organizadas por temática e/ou competição.

## Organização

As soluções estão organizadas em subpastas por temática ou competição específica:

```
Soluções/
├── [nome-da-competição-ou-tema]/
│   ├── problema-a.cpp     
│   ├── problema-b.cpp     
│   └── ...
└── ...
```

## Linguagens Utilizadas

As soluções podem estar implementadas em:
- C++
- Python
- Java
- Outras linguagens conforme necessário (Exemplo: Kotlin)
